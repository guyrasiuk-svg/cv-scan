# CV_SCAN 🔍

אפליקציה לסריקה ומיון קורות חיים עם AI.

---

## 🚀 הפעלה ב-5 דקות

### שלב 1 — הכנת הקבצים

1. הורד את כל תיקיית הפרויקט (`cv-scan`)
2. ודא שהמבנה נראה כך:
```
cv-scan/
  src/
    App.js
    index.js
  public/
    index.html
  package.json
  README.md
```

### שלב 2 — פריסה ב-Vercel (בחינם)

**אפשרות א' — הכי פשוטה: גרור ושחרר**

1. כנס לאתר: https://vercel.com
2. הירשם בחינם (עם Google או GitHub)
3. לחץ **"Add New Project"**
4. גרור את תיקיית `cv-scan` לחלון
5. לחץ **"Deploy"**
6. תוך 2 דקות תקבל קישור לאפליקציה שלך!

**אפשרות ב' — דרך GitHub**

1. צור חשבון GitHub: https://github.com
2. צור repository חדש ושמו `cv-scan`
3. העלה את הקבצים ל-repository
4. כנס ל-Vercel וחבר את ה-repository
5. לחץ Deploy
